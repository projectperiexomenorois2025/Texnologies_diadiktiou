Βοήθεια Χρήσης

Ξεκινώντας με το Streamify

Το Streamify είναι μια πλατφόρμα που σας επιτρέπει να δημιουργείτε, να μοιράζεστε και να παρακολουθείτε λίστες αναπαραγωγής με βίντεο από το YouTube. Ακολουθούν οι βασικές πληροφορίες για να ξεκινήσετε:

Εγγραφή και σύνδεση

Για να χρησιμοποιήσετε όλες τις λειτουργίες του Streamify, πρέπει να δημιουργήσετε έναν λογαριασμό:

  1. Πατήστε το κουμπί "Εγγραφή" στο πάνω μέρος της σελίδας
  2. Συμπληρώστε τη φόρμα με τα απαιτούμενα στοιχεία
  3. Αφού ολοκληρώσετε την εγγραφή, συνδεθείτε με το όνομα χρήστη και τον κωδικό σας

Περιήγηση στον ιστότοπο

Το Streamify έχει ένα απλό και φιλικό μενού πλοήγησης που σας επιτρέπει να μεταβείτε στις εξής κύριες ενότητες:

  • Αρχική: Η κεντρική σελίδα του ιστοτόπου
  • Λίστες: Δείτε δημόσιες λίστες αναπαραγωγής και δημιουργήστε τις δικές σας
  • Προφίλ: Διαχειριστείτε το προφίλ σας και δείτε τις λίστες σας
  • Αναζήτηση: Αναζητήστε περιεχόμενο και άλλους χρήστες

Διαχείριση λιστών αναπαραγωγής

Δημιουργία λίστας αναπαραγωγής

  1. Συνδεθείτε στον λογαριασμό σας
  2. Πηγαίνετε στη σελίδα "Λίστες" και πατήστε "Δημιουργία νέας λίστας"
  3. Συμπληρώστε τον τίτλο της λίστας
  4. Επιλέξτε αν θέλετε η λίστα να είναι δημόσια (ορατή σε όλους) ή ιδιωτική (ορατή μόνο σε εσάς και τους ακολούθους σας)
  5. Πατήστε "Δημιουργία" για να ολοκληρώσετε

Προσθήκη βίντεο σε λίστα

  1. Ανοίξτε τη λίστα που θέλετε να επεξεργαστείτε
  2. Πατήστε "Προσθήκη βίντεο"
  3. Αναζητήστε βίντεο χρησιμοποιώντας το πεδίο αναζήτησης
  4. Πατήστε "Προσθήκη στη λίστα" δίπλα στο βίντεο που θέλετε να προσθέσετε

Επεξεργασία λίστας

  1. Ανοίξτε τη λίστα αναπαραγωγής που θέλετε να επεξεργαστείτε
  2. Πατήστε "Επεξεργασία λίστας"
  3. Αλλάξτε τον τίτλο ή τις ρυθμίσεις απορρήτου (δημόσια/ιδιωτική)
  4. Αφαιρέστε βίντεο από τη λίστα αν χρειάζεται
  5. Πατήστε "Ενημέρωση" για να αποθηκεύσετε τις αλλαγές

Διαγραφή λίστας

  1. Ανοίξτε τη λίστα αναπαραγωγής που θέλετε να διαγράψετε
  2. Πατήστε "Επεξεργασία λίστας"
  3. Κάτω από τις επιλογές επεξεργασίας, θα δείτε την επιλογή "Διαγραφή λίστας"
  4. Πατήστε "Διαγραφή λίστας" και επιβεβαιώστε την επιλογή σας

Παρακολούθηση βίντεο

Αναπαραγωγή λίστας

Για να παρακολουθήσετε τα βίντεο σε μια λίστα αναπαραγωγής:

  1. Ανοίξτε τη λίστα που θέλετε να παρακολουθήσετε
  2. Το πρώτο βίντεο στη λίστα θα φορτωθεί αυτόματα στον ενσωματωμένο YouTube player
  3. Χρησιμοποιήστε τα κουμπιά ελέγχου για αναπαραγωγή, παύση, προηγούμενο ή επόμενο βίντεο
  4. Πατήστε σε οποιοδήποτε βίντεο στη δεξιά πλευρά της οθόνης για να το αναπαράγετε

Αυτόματη αναπαραγωγή

Το Streamify αναπαράγει αυτόματα το επόμενο βίντεο στη λίστα όταν ολοκληρωθεί το τρέχον. Μπορείτε να επιλέξετε οποιοδήποτε βίντεο από τη λίστα για να το αναπαράγετε εκτός σειράς.

Κοινωνικές λειτουργίες

Ακολούθηση χρηστών

Ακολουθώντας άλλους χρήστες, μπορείτε να δείτε τις δημόσιες λίστες τους στο προφίλ σας:

  1. Επισκεφθείτε το προφίλ ενός χρήστη (μέσω αναζήτησης ή πατώντας το όνομά του σε μια λίστα)
  2. Πατήστε το κουμπί "Ακολούθησε" στο προφίλ του
  3. Οι δημόσιες λίστες του χρήστη θα εμφανιστούν στην ενότητα "Ακολουθείτε" στο προφίλ σας

Διαχείριση ακολούθων

Για να δείτε τους χρήστες που ακολουθείτε ή αυτούς που σας ακολουθούν:

  1. Πηγαίνετε στο προφίλ σας
  2. Στην ενότητα "Ακολουθεί", θα δείτε τους χρήστες που ακολουθείτε
  3. Στην ενότητα "Ακόλουθοι", θα δείτε τους χρήστες που σας ακολουθούν

Κοινή χρήση λιστών

Για να μοιραστείτε μια λίστα αναπαραγωγής:

  1. Ανοίξτε τη λίστα που θέλετε να μοιραστείτε
  2. Αντιγράψτε το URL από τη γραμμή διευθύνσεων του περιηγητή σας
  3. Στείλτε αυτό το URL σε φίλους ή δημοσιεύστε το στα μέσα κοινωνικής δικτύωσης

Προσαρμογή και ρυθμίσεις

Αλλαγή θέματος

Το Streamify υποστηρίζει ανοιχτό και σκοτεινό θέμα:

  1. Πατήστε το εικονίδιο του ήλιου/φεγγαριού στο πάνω μέρος της σελίδας
  2. Το θέμα θα αλλάξει αμέσως
  3. Η προτίμησή σας αποθηκεύεται σε cookie και θα εφαρμοστεί σε όλες τις επόμενες επισκέψεις σας

Επεξεργασία προφίλ

Για να ενημερώσετε τα στοιχεία του προφίλ σας:

  1. Πηγαίνετε στο προφίλ σας
  2. Πατήστε "Επεξεργασία προφίλ"
  3. Ενημερώστε τα στοιχεία σας όπως επιθυμείτε
  4. Πατήστε "Αποθήκευση" για να εφαρμόσετε τις αλλαγές

Συχνές ερωτήσεις

Ποιος μπορεί να δει τις λίστες μου;

Οι δημόσιες λίστες είναι ορατές σε όλους τους χρήστες. Οι ιδιωτικές λίστες είναι ορατές μόνο σε εσάς και στους χρήστες που σας ακολουθούν.

Πώς μπορώ να αλλάξω τον κωδικό μου;

Πηγαίνετε στις ρυθμίσεις του λογαριασμού σας και επιλέξτε "Αλλαγή κωδικού".

Ποιος είναι ο μέγιστος αριθμός βίντεο σε μια λίστα;

Δεν υπάρχει περιορισμός στον αριθμό των βίντεο που μπορείτε να προσθέσετε σε μια λίστα.

Υπάρχει εφαρμογή για κινητά;

Προς το παρόν, το Streamify είναι διαθέσιμο μόνο ως διαδικτυακή εφαρμογή. Ωστόσο, ο ιστότοπος είναι πλήρως προσαρμοστικός και λειτουργεί άψογα σε κινητές συσκευές.

Τι συμβαίνει αν ένα βίντεο αφαιρεθεί από το YouTube;

Αν ένα βίντεο αφαιρεθεί από το YouTube, θα παραμείνει στη λίστα σας αλλά δεν θα μπορεί να αναπαραχθεί. Θα εμφανιστεί ένα μήνυμα σφάλματος και η αναπαραγωγή θα προχωρήσει στο επόμενο βίντεο.

Δεν βρήκατε αυτό που ψάχνατε;
